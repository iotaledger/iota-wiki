You may want to melt some of your circulating native tokens. To do so, you will need to call the
`Account.melt_native_token(token_id, melt_amount, options)` function.

If you melt your native tokens, you can still destroy the foundry that created them.

## Code Example

The following example will:

1. Create a wallet.
2. Get Alice's account which was [created in the first guide](../accounts-and-addresses/create-account.mdx).
3. Get the account's balance.
4. Melt 10 native tokens of the first ID.
