An [`Account`](../../references/nodejs/classes/Account/) is a fundamental component of the protocol that allows users
to manage their digital assets and interact with the network. It serves as a container for holding tokens and other
assets, enabling users to send and receive transactions, and engage in various activities within the IOTA ecosystem.

<TipAccountApproaches />

## Code Example

To create an account you need to:

<Tabs groupId="language" queryString>
<TabItem value="rust" label="Rust">

1. Instantiate a [`Wallet`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html) and use it to store
   the [Stronghold](https://wiki.iota.org/stronghold.rs/welcome) mnemonic.

```rust reference
https://github.com/iotaledger/iota-sdk/blob/develop/sdk/examples/how_tos/accounts_and_addresses/create_account.rs#L27-L47
```

2. Create an [`Account`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html) with the alias
   "Alice" by calling the [`Wallet.create_account()`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html#method.create_account)
   function.

```rust reference
https://github.com/iotaledger/iota-sdk/blob/develop/sdk/examples/how_tos/accounts_and_addresses/create_account.rs#L50
```

</TabItem>
<TabItem value="typescript-node" label="Typescript (Node.js)">

1. Instantiate a [`Wallet`](../../references/nodejs/classes/Wallet/) and use it to store the
   [Stronghold](https://wiki.iota.org/stronghold.rs/welcome) mnemonic.

```typescript reference
https://github.com/iotaledger/iota-sdk/blob/develop/bindings/nodejs/examples/how_tos/accounts_and_addresses/create-account.ts#L35-L54
```

2. Create an [`Account`](../../references/nodejs/classes/Account/) with the alias "Alice" by calling the
   [`Wallet.createAccount()`](../../references/nodejs/classes/Wallet/#createAccount)
   function.

```typescript reference
https://github.com/iotaledger/iota-sdk/blob/develop/bindings/nodejs/examples/how_tos/accounts_and_addresses/create-account.ts#L57-L59
```

</TabItem>
<TabItem value="python" label="Python">

1. Instantiate a [`Wallet`](../../references/python/iota_sdk/wallet/) and use it to store the
   [Stronghold](https://wiki.iota.org/stronghold.rs/welcome) mnemonic.

```python reference
https://github.com/iotaledger/iota-sdk/blob/develop/bindings/python/examples/how_tos/accounts_and_addresses/create_account.py#L13-L32
```

2. Create an [`Account`](../../references/python/iota_sdk/wallet/account/) with the alias
   "Alice" by calling the [`Wallet.create_account()`](../../references/python/iota_sdk/wallet/#create_account)
   function.

```python reference
https://github.com/iotaledger/iota-sdk/blob/develop/bindings/python/examples/how_tos/accounts_and_addresses/create_account.py#L34L25
```

</TabItem>
</Tabs>
