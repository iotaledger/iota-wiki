You can retrieve the balance for any [account you created](./create-account.mdx) by calling the
[`Account.getBalance()`](../../references/nodejs/classes/Account/#getbalance) function. Bear in mind that you should
always [`sync`](references/nodejs/classes/Account/#sync) the [`Account`](../../references/nodejs/classes/Account/)
with the node to get the latest outputs. The sync function also returns the balance.

<TipFaucet />

## Code Example

To get an account balance you will need to:

<Tabs groupId="language" queryString>
<TabItem value="rust" label="Rust">

1. Instantiate a [`Wallet`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html), get
   Alice's `Account` which was
   [created in the first guide](../accounts-and-addresses/create-account.mdx).
2. Sync the account with the node to get the latest transactions and the account's
   [balance](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/account/struct.Account.html#method.balance) by calling
   the [`Account.sync()`](https://docs.rs/iota-sdk/latest/iota_sdk//wallet/account/struct.Account.html#method.sync)
   function.

```rust reference
https://github.com/iotaledger/iota-sdk/blob/develop/sdk/examples/how_tos/accounts_and_addresses/check_balance.rs#L28
```

</TabItem>
<TabItem value="typescript-node" label="Typescript (Node.js)">

1. Instantiate a [`Wallet`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html), Get
   Alice's `Account` which was

2. Sync the account with the node to get the latest transactions and the account's

   function.

</TabItem>

1. Instantiate a [`Wallet`](https://docs.rs/iota-sdk/latest/iota_sdk/wallet/core/struct.Wallet.html), Get
