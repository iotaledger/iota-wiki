Once you have [created your account](../accounts_and_addresses/create_account.mdx) and
[requested funds from the faucet](./request_funds.mdx) you are ready to send your first
transaction.

<TipFaucet />

## Code Example

The following example will:

1. Create a wallet.
2. Get Alice's account which was [created in the first guide](../accounts_and_addresses/create_account.mdx).
3. Send 1 SMR to an address using the `Account.send_amount(addresses_with_amount, options)` function. Make sure to update the code sample to send to the address of your choice.
